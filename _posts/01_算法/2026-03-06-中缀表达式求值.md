当表达式以字符串形式给出时，例如3*(4+5)-2，计算机如何正确处理运算符优先级、括号以及负号？

---

## 1. 问题描述

在编程中，经常会遇到需要 **计算字符串表达式** 的问题，例如：

```
3 + 4 * 2 / (1 - 5)
```

或者

```
3 * (-2) + 5
```

这类表达式通常包含：

* 数字（可能是多位数）
* 运算符 `+ - * /`
* 括号 `()`
* 一元负号（如 `-2`）

这类问题在算法题中也非常常见，这些题本质上都是 **表达式解析与求值问题**。

本文将介绍两种经典算法：

1. **中缀表达式 → 后缀表达式 → 求值**
2. **双栈算法（直接计算）**

这两种方法都源于 **Shunting Yard Algorithm**，由著名计算机科学家 **Edsger W. Dijkstra** 提出。

---

## 2. 前置知识：表达式表示法

常见表达式表示法：

| 表达式类型 | 示例      |
| ----- | ------- |
| 中缀表达式 | `3 + 4` |
| 前缀表达式 | `+ 3 4` |
| 后缀表达式 | `3 4 +` |

我们日常书写的是 **中缀表达式**。

但计算机更容易处理的是 **后缀表达式（逆波兰表达式）**。

例如：

```
3 + 4 * 2
```

对应后缀表达式：

```
3 4 2 * +
```

后缀表达式的优势：

* 不需要括号
* 不需要考虑优先级
* 可以用栈线性计算

---

## 3. 算法一：中缀表达式 → 后缀表达式 → 求值

### 3.1. 中缀表达式 → 后缀表达式

**核心数据结构：**

```
输出序列
运算符栈
```

**规则总结：**

假设表达式为：

```
3 + 4 * 2 / (1 - 5)
```

依次扫描表达式，分情况讨论。

👉 情况1：数字，直接加入输出序列。

👉 情况2：左括号，直接加入运算符栈。

👉 情况3：右括号，弹出运算符直到遇到 `(`；然后丢弃 `(`。---

👉 情况4：运算符，假设当前运算符为 `op`，处理规则如下：

```
while 栈顶运算符优先级 >= 当前运算符
    pop到输出

当前运算符入栈
```

👉 情况5：表达式结束，把栈中运算符全部弹出，放入到输出序列。

👉 特殊情况：一元负号处理

表达式中的 `-` 可能表示：

```
3 - 2   （二元运算符）
-2      （一元运算符）
3 * (-2)
```

解决方法：

```
-2 → 0 - 2
```

当 `-` 出现在以下情况时认为是一元负号：

* 表达式开头
* 前一个字符是 `(` 或运算符

---

**完整转换示例**

表达式：

```
3 + 4 * 2 / (1 - 5)
```

过程：

| 扫描 | 输出                | 栈       |
| -- | ----------------- | ------- |
| 3  | 3                 |         |
| +  | 3                 | +       |
| 4  | 3 4               | +       |
| *  | 3 4               | + *     |
| 2  | 3 4 2             | + *     |
| /  | 3 4 2 *           | + /     |
| (  | 3 4 2 *           | + / (   |
| 1  | 3 4 2 * 1         | + / (   |
| -  | 3 4 2 * 1         | + / ( - |
| 5  | 3 4 2 * 1 5       | + / ( - |
| )  | 3 4 2 * 1 5 -     | + /     |
| 结束 | 3 4 2 * 1 5 - / + |         |

最终：

```
3 4 2 * 1 5 - / +
```

---

**中缀转后缀代码**

```python
def infix_to_postfix(expr: str):

    priority = {'+':1,'-':1,'*':2,'/':2}

    output = []
    ops = []

    i = 0
    n = len(expr)

    prev_token = None

    while i < n:

        c = expr[i]

        if c == ' ':
            i += 1
            continue

        # 读取数字（多位）
        if c.isdigit():

            num = 0
            while i < n and expr[i].isdigit():
                num = num*10 + int(expr[i])
                i += 1

            output.append(str(num))
            prev_token = "num"
            continue

        # 左括号
        if c == '(':
            ops.append(c)
            prev_token = "("

        # 右括号
        elif c == ')':

            while ops[-1] != '(':
                output.append(ops.pop())

            ops.pop()
            prev_token = ")"

        # 运算符
        elif c in "+-*/":

            # 一元负号
            if c == '-' and (prev_token in (None,"(","op")):
                output.append("0")

            while (ops and ops[-1] != '(' and
                   priority[ops[-1]] >= priority[c]):
                output.append(ops.pop())

            ops.append(c)
            prev_token = "op"

        i += 1

    while ops:
        output.append(ops.pop())

    return output
```

### 3.2. 后缀表达式求值

后缀表达式计算非常简单。

规则：

```
数字 → 入栈
运算符 → 弹出两个数计算
```

代码：

```python
def eval_postfix(tokens):

    stack = []

    for t in tokens:

        if t not in "+-*/":
            stack.append(int(t))

        else:

            b = stack.pop()
            a = stack.pop()

            if t == '+':
                stack.append(a+b)
            elif t == '-':
                stack.append(a-b)
            elif t == '*':
                stack.append(a*b)
            else:
                stack.append(int(a/b))

    return stack[0]
```

## 4. 算法二：双栈算法

除了先转换表达式，还可以 **边扫描边计算**。

核心思想：

```
两个栈
```

```
nums  → 数字栈
ops   → 运算符栈
```

---

扫描表达式：

👉 情况1：读取数字

入 `nums` 栈。

---

👉 情况2：左括号

入 `ops` 栈。

---

👉 情况3：右括号

不断计算直到遇到 `(`。

---

👉 情况4：运算符

若当前运算符 `op`：

```
while 栈顶优先级 >= op
    先计算
```

然后 `op` 入栈。

---

👉 计算函数

每次计算：

```
b = nums.pop()
a = nums.pop()
op = ops.pop()

result = a op b
nums.push(result)
```

---

👉 双栈实现代码

```python
def calculate(expr):

    nums = []
    ops = []

    priority = {'+':1,'-':1,'*':2,'/':2}

    def calc():

        b = nums.pop()
        a = nums.pop()
        op = ops.pop()

        if op == '+':
            nums.append(a+b)
        elif op == '-':
            nums.append(a-b)
        elif op == '*':
            nums.append(a*b)
        else:
            nums.append(int(a/b))

    i = 0
    n = len(expr)
    prev = None

    while i < n:

        c = expr[i]

        if c == ' ':
            i += 1
            continue

        # 数字
        if c.isdigit():

            num = 0
            while i<n and expr[i].isdigit():
                num = num*10 + int(expr[i])
                i += 1

            nums.append(num)
            prev = "num"
            continue

        # 左括号
        if c == '(':
            ops.append(c)
            prev = "("

        # 右括号
        elif c == ')':

            while ops[-1] != '(':
                calc()

            ops.pop()
            prev = ")"

        # 运算符
        elif c in "+-*/":

            if c == '-' and prev in (None,"(","op"):
                nums.append(0)

            while (ops and ops[-1] != '(' and
                   priority[ops[-1]] >= priority[c]):
                calc()

            ops.append(c)
            prev = "op"

        i += 1

    while ops:
        calc()

    return nums[-1]
```

## 5. 两种算法对比

| 算法      | 特点         |
| ------- | ---------- |
| 中缀 → 后缀 | 逻辑清晰，易理解   |
| 双栈算法    | 实现更直接，代码更短 |

区别：

```
方法一
先转换表达式
再计算

方法二
扫描时直接计算
```

在实际面试或算法题中：

* **双栈算法更常见**
* **后缀表达式更适合理解原理**

表达式求值是 **编译原理与数据结构结合的经典问题**。

核心思想：

```
利用栈管理运算顺序
```

两种常见方法：

```
中缀 → 后缀 → 求值
双栈算法
```

掌握这两种方法后，可以解决绝大多数 **表达式解析类算法题**。
