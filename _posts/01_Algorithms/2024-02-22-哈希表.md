在算法题中，如果还在用 O(n²) 两重循环找匹配，那通常意味着：这题该用哈希表了。
它的核心价值只有一句话：
> **用空间换时间，实现 O(1) 级别的查询。**

---

## 基本概念

**哈希表（Hash Table）**是一种基于 **哈希函数（Hash Function）** 的数据结构，通过将元素映射到数组下标位置，实现**近乎 O(1)** 的插入、删除和查找操作。

> 通过「哈希函数」把 key 映射到数组下标，从而实现快速存取的数据结构。

简单理解就是：

```python
dict[key] = value
```

底层流程：

```
key → hash函数 → 数组下标 → 存储位置
```

---

**特点**

* **查找快**：平均时间复杂度 O(1)。
* **无序性**：元素在哈希表中通常是无序的。
* **键唯一**：哈希表的 key 是唯一的。

**核心能力**

1. **判断是否存在**
2. **统计出现次数**
3. **记录某个值对应的位置**

---

**常见实现方式**

1. 数组 + 链表（拉链法 / 分离链接法）
   * 每个数组位置存放一个链表，冲突的元素链在一起。
   * 适合频繁插入删除。

2. 数组 + 开放定址（Open Addressing）
   * 如果位置被占用，就向后找空位置。
   * 节省指针空间，但删除复杂。

---

**常见操作（Python 示例）**
```python
# Python 内置 dict 就是哈希表
hash_map = {}

# 插入 / 更新
hash_map['apple'] = 3
hash_map['banana'] = 5

# 查找
if 'apple' in hash_map:
    print(hash_map['apple'])  # 输出 3

# 删除
del hash_map['banana']

# 遍历
for key, value in hash_map.items():
    print(key, value)
```

## 经典模板题型

---

### 模型一：两数之和

```python
def twoSum(nums, target):
    mp = {}
    for i, x in enumerate(nums):
        if target - x in mp:
            return [mp[target - x], i]
        mp[x] = i
```

核心思想：

> 一边遍历，一边记录“之前出现过的数”。

---

### 模型二：计数型

```python
from collections import Counter
cnt = Counter(nums)
```

用途：

* 判断是否存在重复
* 出现次数统计
* 多数元素问题

---

### 模型三：前缀和 + 哈希

例如：子数组和为 K

```python
def subarraySum(nums, k):
    mp = {0: 1}
    s = 0
    ans = 0
    for x in nums:
        s += x
        ans += mp.get(s - k, 0)
        mp[s] = mp.get(s, 0) + 1
    return ans
```

核心思想：

> 如果 s[i] - s[j] = k
> 那么只要之前出现过 s[i] - k

这是面试高频模型。

---

### 模型四：滑动窗口 + 哈希

例如：

* 最长无重复子串

```python
def lengthOfLongestSubstring(s):
    seen = {}
    left = 0
    ans = 0
    for right, ch in enumerate(s):
        if ch in seen and seen[ch] >= left:
            left = seen[ch] + 1
        seen[ch] = right
        ans = max(ans, right - left + 1)
    return ans
```

哈希用来：

> 记录字符上一次出现位置

## 哈希表常见优化技巧

### 使用 defaultdict

```python
from collections import defaultdict
mp = defaultdict(int)
```

避免写：

```python
if key in mp:
```

---

### 使用 Counter

```python
from collections import Counter
cnt = Counter(nums)
```

统计问题首选。

---

### 使用 set

只需要判断是否存在时：

```python
s = set(nums)
```

比 dict 更轻量。

## 其他知识点

👉 **什么时候该想到哈希表？**

关键词：

* 两数之和
* 是否存在
* 出现次数
* 子数组和
* 去重
* 找重复
* 前缀和差值

---

👉 **哈希表 vs 排序**

很多题可以：

* 排序 + 双指针（O(n log n)）
* 哈希表（O(n)）

一般优先哈希。

但排序适用于：

* 需要顺序
* 需要去重后的组合问题

---

👉 **常见错误**

* ❌ 忘记初始化 `{0:1}`（前缀和问题）
* ❌ 更新顺序写错
* ❌ 频率统计漏减

---

👉 **哈希表在 DP 中的作用**

高级应用：

* 状态压缩 DP
* 记忆化搜索
* 子序列计数
* 动态维护状态

哈希是很多高阶算法的“隐形支撑”。
