    Python列表操作技巧

    On this page

    • 列表基本操作
    • 切片操作
    • 删除列表元素

    Python 中的 列表（List） 是一种有序、可变、可重复的序列类型，用于存储一组项目。它是 Python 中最常用的数据结构之一，类似于数组，但功能更强大。

    列表基本操作

    创建列表

    # 空列表
a = []

# 含元素的列表
b = [1, 2, 3]

# 混合类型
c = [1, "hello", True, 3.14]

# 嵌套列表（二维）
d = [[1, 2], [3, 4]]

    列表推导式

    squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]
evens = [x for x in lst if x % 2 == 0]

    访问元素

    lst = [10, 20, 30, 40]

print(lst[0])    # 10
print(lst[-1])   # 40（最后一个）

    修改元素

    lst[1] = 200
print(lst)  # [10, 200, 30, 40]

    添加元素

    lst.append(50)        # 末尾添加一个元素
lst.extend([60, 70])  # 扩展多个元素
lst.insert(1, 999)    # 在索引1插入一个元素

    删除元素

    del lst[2]         # 删除索引为2的元素
lst.remove(999)    # 删除第一个值为999的元素
lst.pop()          # 删除最后一个元素并返回
lst.clear()        # 清空列表

    清空列表

    lst = [1, 2, 3]
lst.clear()  # 清空列表
print(lst)  # []

    查询与判断

    len(lst)           # 列表长度
20 in lst          # 是否包含某个元素（True/False）
lst.index(30)      # 查找元素索引
lst.count(20)      # 某元素出现次数

    排序与反转

    lst.sort()         # 原地排序
lst.sort(reverse=True)  # 降序
lst.reverse()      # 反转列表

    列表切片

    lst = [0, 1, 2, 3, 4, 5]
print(lst[1:4])    # [1, 2, 3]
print(lst[::-1])   # [5, 4, 3, 2, 1, 0] 倒序

    遍历列表

    for item in lst:
    print(item)

# 同时获取索引和值
for idx, val in enumerate(lst):
    print(idx, val)

    列表是可变对象（mutable）

    a = [1, 2, 3]
b = a
b[0] = 100
print(a)  # [100, 2, 3]，b 修改会影响 a（指向同一对象）

# 正确复制方法
b = a[:]  # 或 list(a)、copy.copy(a)

    切片操作

    切片是 Python 中提取列表（或字符串、元组）一部分的强大方式，语法如下。

    list[start:stop:step]
    • start: 起始索引（包含）
    • stop: 结束索引（不包含）
    • step: 步长（可选）

    基本示例

    lst = [0, 1, 2, 3, 4, 5, 6]
print(lst[1:4])    # [1, 2, 3]  从索引1到3
print(lst[:3])     # [0, 1, 2]  从头到索引2
print(lst[3:])     # [3, 4, 5, 6] 从索引3到结尾
print(lst[:])      # [0, 1, 2, 3, 4, 5, 6] 整个列表的复制

    步长 step

    lst = [0, 1, 2, 3, 4, 5, 6]
print(lst[::2])    # [0, 2, 4, 6] 每隔一个取一个（偶数索引）
print(lst[1::2])   # [1, 3, 5]    从索引1开始，每隔一个取（奇数索引）

    倒序切片（step 为负数）

    lst = [0, 1, 2, 3, 4, 5]
print(lst[::-1])   # [5, 4, 3, 2, 1, 0]  反转列表
print(lst[4:1:-1]) # [4, 3, 2] 从索引4到2（不包含1），倒着来

    切片赋值（修改）

    lst = [1, 2, 3, 4, 5]
lst[1:4] = [20, 30, 40]  # 替换索引1到3的值
print(lst)  # [1, 20, 30, 40, 5]

    删除切片元素

    lst = [1, 2, 3, 4, 5]
del lst[1:4]  # 删除索引1到3
print(lst)  # [1, 5]

    切片复制列表（浅拷贝）

    a = [1, 2, 3]
b = a[:]  # 浅复制
b[0] = 100
print(a)  # [1, 2, 3]
print(b)  # [100, 2, 3]

    索引支持负数

    lst = [10, 20, 30, 40, 50]
print(lst[-3:-1])  # [30, 40] 倒数第3到倒数第2（不包含最后一个）

    删除列表元素

    在 Python 中，删除列表中的元素有多种方式，诸如按值删除、按索引删除、条件删除，还是就地修改原列表。

    按索引删除

    使用 del 或 pop()：

    # del 删除指定索引元素
lst = [1, 2, 3, 4, 5]
del lst[2]  # 删除索引为 2 的元素，即 3
print(lst)  # [1, 2, 4, 5]

# pop() 删除并返回指定索引元素（默认最后一个）
x = lst.pop(1)  # 删除索引 1 的元素，即 2
print(x)   # 2
print(lst) # [1, 4, 5]

    按切片删除

    lst = [10, 20, 30, 40, 50]
del lst[1:3]  # 删除索引 1 和 2，即 20 和 30
print(lst)  # [10, 40, 50]

# 带步长删除
lst = [0, 1, 2, 3, 4, 5]
del lst[::2]  # 删除索引为 0, 2, 4 的元素
print(lst)  # [1, 3, 5]

    语法形式是：

    del lst[start:stop]
    • start：起始索引（包含）
    • stop：结束索引（不包含）
    • 可以带步长 step，例如 del lst[::2] 删除偶数索引项。

    注意事项：

    • del 是原地修改列表，不返回新列表。
    • 尽量避免在遍历中对原列表使用 del，以免逻辑混乱。

    按值删除

    使用 remove()（只会删掉第一个匹配值）：

    lst = [1, 2, 3, 2, 4]
lst.remove(2)  # 删除第一个 2
print(lst)     # [1, 3, 2, 4]

    条件删除（保留不符合条件的）

    • 使用列表推导式创建一个新列表 
      lst = [1, 2, 3, 4, 5, 6]
lst = [x for x in lst if x % 2 == 0]  # 只保留偶数
print(lst)  # [2, 4, 6]
    • 或者使用 filter() 
      lst = list(filter(lambda x: x % 2 == 0, lst))
    • 注意： 不要在遍历列表时直接删除元素，否则可能跳过部分元素。如下为错误示例。 
      lst = [1, 2, 3, 4]
for x in lst:
  if x % 2 == 0:
      lst.remove(x)  # ⚠️ 错误写法
print(lst)  # [1, 3]，2 被删后，3 位置变化，跳过了
