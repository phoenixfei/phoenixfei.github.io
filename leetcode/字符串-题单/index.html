    字符串是算法中非常基础且高频考察的类型，涵盖了模式匹配、字符串哈希、双指针、前缀和、动态规划等多种技术。

    字符串是算法中非常基础且高频考察的类型，涵盖了模式匹配、字符串哈希、双指针、前缀和、动态规划等多种技术。以下是系统的字符串专题整理，附带子分类、核心算法说明与典型题目链接，便于系统学习。

    字符串专题分类（附典型题目）

    1. 基础操作 & 双指针

    • 描述：字符串反转、判断回文、去重等基础题。
    • 技巧：双指针、滑动窗口。
    • 🔗 Leetcode 344. 反转字符串
    • 🔗 Leetcode 125. 验证回文串

    2. 滑动窗口

    • 描述：处理子串问题，常用于最长/最短子串、包含/匹配问题。
    • 技巧：左右指针移动、哈希计数。
    • 🔗 Leetcode 3. 无重复字符的最长子串
    • 🔗 Leetcode 76. 最小覆盖子串

    3. 字符串匹配

    • 描述：在文本中查找模式串。
    • 技巧：KMP、Rabin-Karp、Boyer-Moore。
    • 🔗 Leetcode 28. 找出字符串中第一个匹配项的下标（KMP）
    • 🔗 AcWing 831. KMP字符串

    4. 前缀和与哈希

    • 描述：快速判断子串出现次数、相同字符集等。
    • 技巧：前缀哈希、字符计数数组。
    • 🔗 Leetcode 560. 和为K的子数组（数组版）
    • 🔗 Leetcode 438. 找到字符串中所有字母异位词

    5. 回文串相关

    • 描述：判断是否为回文串，找最长回文子串、回文划分。
    • 技巧：中心扩展、动态规划、Manacher。
    • 🔗 Leetcode 5. 最长回文子串
    • 🔗 Leetcode 131. 分割回文串

    6. 字符串哈希

    • 描述：快速判断两个子串是否相等。
    • 技巧：滚动哈希（Rabin-Karp）、双哈希防冲突。
    • 🔗 AcWing 841. 字符串哈希
    • 🔗 Leetcode 1044. 最长重复子串

    7. 字典树（Trie）

    • 描述：多字符串高效插入与查找，常用于前缀匹配。
    • 技巧：树结构、哈希表/数组实现。
    • 🔗 Leetcode 208. 实现 Trie
    • 🔗 Leetcode 648. 单词替换

    8. 分组变形题

    • 描述：字母异位词分组、相同结构字符分组。
    • 技巧：计数排序、哈希编码。
    • 🔗 Leetcode 49. 字母异位词分组

    9. 动态规划（高级）

    • 描述：求两个字符串之间的最小变换（编辑距离等）。
    • 技巧：二维 DP。
    • 🔗 Leetcode 72. 编辑距离
    • 🔗 Leetcode 115. 不同的子序列
